FoodFinder Ui Design

Web Application to search food online

About the project

A company asked for the UI Design for a restaurant search platform they want to developed. The client wanted to show the users a way to search for restaurants compare them by price, location and type of food.

Tools

Sketch + Principle
InVision
User Interface Design
Web Design

As a Ui Designer I was asked to develope the early stage of the ui style guide and the webapp desktop designs. The client wanted to consider the platform would be later take to a second stage where the iOS and Android apps would be develped and designed. The feature I was asked to design where:

  • Filters
  • Maps
  • Restaurant information
  • Raiting
  • Price

Also, I got to work with the animation of the components. This are some of the results for the ui design and how they interact with the users.
